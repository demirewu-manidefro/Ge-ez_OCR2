About Ge'ez OCR

Project Overview

This project aims to digitize Ge'ez manuscripts using modern AI technology. Ge'ez is an ancient Semitic language used in Ethiopian and Eritrean religious and historical texts.

How It Works

  • Uses a fine-tuned TrOCR (Transformer-based Optical Character Recognition) model
  • Supports full-page recognition by segmenting pages into individual lines
  • Each line is processed separately for improved accuracy
  • Results are combined with proper line breaks

Technologies Used

  • Flask: Web framework for the application
  • PyTorch: Deep learning framework
  • Transformers: HuggingFace library for state-of-the-art NLP models
  • OpenCV: Computer vision library for image preprocessing and line segmentation
  • ReportLab: PDF generation
